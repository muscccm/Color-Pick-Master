## 📖 简介

如果你是设计师，摄影师，调色师，美术师，或者要做PPT或从事一切需要做颜色搭配的工作的人，那一定经历过在自己的作品上对颜色难以抉择的时刻。
现在你可以通过我们的工具快速提取优秀设计的颜色，快速调色，甚至可以通过内嵌的专业级别的调色工具高效设计自己的配色系统。
无论你是小白，还是专业选手，这款工具都能使你对色彩的把控游刃有余。
更内置强大且专业的色轮以及HSB配色工具，帮助你科学地搭配颜色。
当然，你也完全可以把它当做你学习色彩理论时的学习工具。

## ✨ 主要功能

### 图片颜色提取

* 将优秀作品图片中的配色快速提取出来。支持拖拽，点击，和复制粘贴。可以批量上传。
* 并将提取出来的配色直观展示为颜色条进行观察。
* 并可将提取的的颜色快速选取色值。仅需通过简单的点击即可复制。
* 并且支持将色条合并到图片上进行导出，也能将色条导出为图片进行保存。

### 照片调色

* 我们的调色系统和其他细致调色软件不同，我们的调色系统旨在帮助用户快速确定图片的色彩基调或者快速将其他人的优秀配色快速运用到自己图片中预览。这对摄影师，调色师，剪辑师来说非常有用。
* 支持实时对比原图和成图。
* 支持色彩映射。并可从保存的配色中快速提取配色用于自己的图片。
* 同样支持将配色条和成图合并导出。
* 可以调整配色混合强度。

### 配色模板

* 强大的配色预览系统，内置15种，超全应用场景真实模板。
* 模板都是由html真实组件制作而非简单图片。
* 所有模板都可自由搭配调整颜色。并实时预览。能快速，直观的预览你的配色在不同场景下的应用效果。
* 同样支持从右侧已保存的配色中快速提取。
* 还提供少，中，多三种配色层级可供选择，不仅适配你简约的配色逻辑也同时适配复杂的配色逻辑。
* 一键重置，来复原模板默认配色。

### 强大且专业的调色系统。

* 采用专业软件所常用的HSB调色盘系统对任何地方的色彩进行快速调整。
* 大面积颜色预览。对比某些软件中的超小颜色预览框来说更加直观。
* 除了可事实调整颜色，还可快速复制色值，仅单击即可复制色值，无需像某些软件一样还得选择色值文字复制。
* 全屏幕吸管工具，快速吸取屏幕任何位置的颜色。

### 强大且专业的色轮系统

* 专业色轮帮助你科学进行色彩搭配，内置符合，类比，单色，三原色，补色，分割补色，正方形，复合，浓度，自定义，多种色彩搭配模式。
* 不同模式都会在色轮上产生符合规则的节点。
* 还可将任何颜色作为主色进行任意模式的搭配。
* 一个色轮最多可添加12个颜色节点。
* 还可将配色一键存入备选色组，方便随时调用或吸取。
* 任何地方保存的色组都可以双击一键还原到色轮上。

### 精美主题

* 精心设计的暗色主题和浅色主题。

### 多语言支持

* 程序支持中文，英文，日文三种语言，在程序左下角切换。

- **无损处理**：所有操作不会修改原始图片文件
